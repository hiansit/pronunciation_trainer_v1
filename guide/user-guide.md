## はじめに

### 動作環境

| ブラウザ | TTS (お手本再生) | STT (マイク入力) |
|---------|:---:|:---:|
| **Google Chrome** (推奨) | ✅ | ✅ |
| Microsoft Edge | ✅ | ✅ |
| Firefox | ✅ | ❌ (非対応) |
| Safari (macOS/iOS) | ✅ | ✅ (部分的) |

> ⚠️ **マイク入力は Chrome を強く推奨します。** Firefox は Web Speech API の音声認識に対応していないため、お手本再生のみ利用できます。

### マイクの許可

初めてマイク入力ボタンを押すと、ブラウザからマイク使用の許可を求められます。「許可」を選択してください。

## デッキ管理画面

アプリを開くと最初に表示される画面です。

### デッキの作成

1. 画面右上の **＋ 新規作成** ボタンをクリック
2. デッキ名を入力（例：「英語 日常会話」「TOEIC Part 2」）
3. **保存** をクリック

### 例文の登録（TSVインポート — おすすめ）

大量の例文を一度に登録できます。

1. デッキを選択 → **📋 TSVインポート** をクリック
2. テキストエリアに **タブ区切りテキスト** を貼り付け
3. **インポート** をクリック

```
テキスト(TAB)訳文(TAB)メモ
```

各列の意味：
| 列 | 内容 | 必須 |
|---|------|:---:|
| 1列目 | 練習テキスト（発音の対象） | ✅ |
| 2列目 | 訳文・意味 | 任意 |
| 3列目 | メモ・覚え方 | 任意 |

> 📝 TSVフォーマットの詳細は [TSVフォーマットガイド](tsv-format.md) をご覧ください。

### 例文の手動追加

1. デッキを選択 → **＋ 例文追加** をクリック
2. テキスト・訳文・メモを入力
3. **保存** をクリック

### 例文の編集・削除

各例文の右側にある ✏️（編集）/ 🗑（削除）ボタンから操作できます。

## 練習画面

### 基本の流れ

1. 画面上部のドロップダウンから **デッキを選択**
2. **▶ お手本を聞く** — ブラウザの音声合成で例文を読み上げます
3. **🎤 マイク入力** — あなたの発音を録音・テキスト化します
4. **突き合わせ結果** — 原文との一致度がスコアとして表示されます
5. **レベルを確定して次へ** — 習熟レベルを設定して次の例文に進みます

### 言語の選択

右上の言語セレクタから練習したい言語を選びます。これはTTS（お手本再生）と STT（音声認識）の両方に適用されます。

### 練習モード

| モード | 動作 | おすすめの使い方 |
|-------|------|---------------|
| **順番通り** | 登録順に出題 | 初回学習、系統的な復習に |
| **ランダム** | ランダムに出題 | 刺激を変えたい時に |
| **弱点集中** | 低レベルから優先して出題 | 苦手な例文を集中的に |

### 突き合わせ結果の見方

| 表示色 | 意味 |
|-------|------|
| 🟢 **緑** | 完全一致 — 正しく認識されました |
| 🟡 **オレンジ** | 部分一致 — 近い単語として認識されました |
| 🔴 **赤（取消線）** | 不一致 — 認識されなかった単語 |
| 🟣 **紫（イタリック）** | 余分 — 原文にない単語が認識されました |

### 習熟レベル

スコアに基づいてレベルが自動提案されますが、自分で変更して確定できます。

| Lv | 名前 | 色 | 目安 |
|:--:|------|:--:|-----|
| 0 | 未学習 | ⚪ | まだ練習していない |
| 1 | 練習中 | 🔴 | スコア 1〜49% |
| 2 | やや定着 | 🟠 | スコア 50〜69% |
| 3 | ほぼ定着 | 🟡 | スコア 70〜89% |
| 4 | 習得済み | 🟢 | スコア 90%以上 |

## 進捗一覧画面

デッキを選択すると、学習の進捗が一目でわかります。

### サマリーバー

画面上部に **習得率** と **レベル別の分布バー** が表示されます。

### 統計カード

- 📚 総カード数
- ✏️ 学習済み（1回以上練習したカード）
- 🏆 習得済み（レベル4に到達したカード）
- 🔄 総練習回数
- 📊 平均スコア

### フィルター

フィルターボタンでレベル別に絞り込み表示できます。例文をクリックすると、その例文の練習画面に直接ジャンプします。

## データのバックアップと復元

### JSONエクスポート（バックアップ）

1. デッキ管理画面でデッキを選択
2. **💾 JSONエクスポート** をクリック
3. `.json` ファイルがダウンロードされます

> 💡 このファイルにはデッキ名・全例文・進捗データが含まれます。

### JSONインポート（復元）

1. **📂 JSONインポート** をクリック
2. 以前エクスポートした `.json` ファイルを選択
3. **インポート** をクリック

進捗データも含めて完全に復元されます。別のPCや別のブラウザへのデータ移行にも使えます。

## 学習のヒント

### 効果的な練習サイクル

1. **初回**: 順番通りモードで全例文を一通り練習
2. **復習**: 弱点集中モードで低レベルの例文を重点的に
3. **仕上げ**: ランダムモードで定着度をチェック

### より良いスコアのために

- **静かな環境** でマイクを使う
- **はっきり、ゆっくり** 発音する
- まず **お手本を2〜3回聞いてから** マイク入力する
- 一文が長い場合は **区切って練習** し、慣れたら一気に

## トラブルシューティング

### マイクが反応しない

1. ブラウザのアドレスバー左のアイコン → マイクの許可を確認
2. PC本体のマイク設定を確認
3. 他のアプリがマイクを占有していないか確認

### 「音声が検出されませんでした」と表示される

- マイクの入力音量が小さい可能性があります
- より大きな声で、マイクに近づいて発声してみてください

### お手本の音声が再生されない

- ブラウザの音声出力がミュートになっていないか確認
- 一部の環境では初回操作時にユーザー操作が必要です（自動再生制限）

### データが消えた

- ブラウザの閲覧データ（Cookie・サイトデータ）を削除すると IndexedDB も消えます
- 定期的に **JSONエクスポート** でバックアップを取ることを推奨します
