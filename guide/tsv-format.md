# 📋 TSVフォーマットガイド

このツールでは **TSV（Tab-Separated Values）** 形式で例文を一括インポートできます。

---

## 基本フォーマット

```
テキスト[TAB]訳文[TAB]メモ
```

- 各列は **タブ文字** で区切ります
- 1行につき1つの例文です
- 2列目（訳文）と3列目（メモ）は省略可能です

---

## 入力例

### 2列（テキスト + 訳文）

```
Hello, how are you?	こんにちは、お元気ですか？
I'm fine, thank you.	元気です、ありがとう。
What is your name?	お名前は何ですか？
```

### 3列（テキスト + 訳文 + メモ）

```
Good morning!	おはようございます！	朝の挨拶。カジュアルに "Morning!" でもOK
Thank you very much!	どうもありがとうございます！	感謝の表現
```

### 1列（テキストのみ）

```
Good morning!
Good afternoon!
Good evening!
```

---

## TSVファイルの作り方

### 方法1: テキストエディタで直接書く

メモ帳やVS Codeで、列の間にタブキーを押して区切りを入れます。

### 方法2: Excelやスプレッドシートから作る

1. Excel や Google スプレッドシートで例文リストを作成
2. A列にテキスト、B列に訳文、C列にメモ
3. 全セルを選択してコピー（Ctrl+C）
4. TSVインポート画面に貼り付け（Ctrl+V）

> 💡 スプレッドシートからコピーすると自動的にタブ区切りになります。

### 方法3: `.tsv` ファイルを用意する

テキストファイルの拡張子を `.tsv` にして保存します。文字コードは **UTF-8** を推奨します。

---

## 注意事項

- **空行** は自動的にスキップされます
- テキスト（1列目）が空の行もスキップされます
- タブ文字の代わりにスペースを使うと、1つの列として認識されるのでご注意ください
- セル内の改行には対応していません。1行1例文で記述してください
