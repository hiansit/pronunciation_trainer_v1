# 🎙 発音特訓ツール Pro

**Web Speech API を活用した、ブラウザ完結型の発音練習ツール。**

多数の例文をデッキとして管理し、お手本再生 → マイク入力 → 自動採点 → 進捗管理の流れで体系的に発音トレーニングができます。

> 🌐 **[▶ オンラインデモを試す](https://hiansit.github.io/pronunciation_trainer_v1/)**
> （GitHub Pages にデプロイ済み。インストール不要、登録不要。）

---

## ✨ 特長

| 機能 | 説明 |
|------|------|
| 🔊 **お手本再生** | ブラウザ内蔵の音声合成(TTS)で例文を読み上げ |
| 🎤 **マイク入力** | 音声認識であなたの発音をテキスト化 |
| 📝 **自動採点** | 原文と発音を単語レベルで突き合わせ、正確度スコアを算出 |
| 📚 **デッキ管理** | 例文をデッキにまとめて管理。TSV / JSON でインポート & エクスポート |
| 📊 **進捗可視化** | 5段階レベルの色分け表示、サマリーバー、統計ダッシュボード |
| 🎯 **3つの練習モード** | 順番通り ／ ランダム ／ 弱点集中 |
| 💾 **完全ローカル** | データはブラウザの IndexedDB に保存。サーバー送信なし、登録不要 |
| 📱 **レスポンシブ** | PC でもスマホでも快適に使える |

---

## 🚀 使い方（かんたん3ステップ）

### 1. デッキを作る
「デッキ管理」タブで **＋ 新規作成** → デッキ名を入力して保存。

### 2. 例文を登録
作成したデッキを選択 → **TSVインポート** で一括登録、または **＋ 例文追加** で1件ずつ追加。

### 3. 練習する
「練習」タブに切り替え → デッキを選択 → **▶ お手本を聞く** → **🎤 マイク入力** → 結果を確認してレベルを記録！

> 📖 詳しい操作方法は [ユーザーガイド](guide/user-guide.md) をご覧ください。

## 🛠 技術仕様

- **フレームワーク不使用** — ピュアHTML / CSS / JavaScript
- **依存パッケージゼロ** — npm不要
- **IndexedDB** — ブラウザ内にデータを永続化
- **Web Speech API** — SpeechSynthesis (TTS) + SpeechRecognition (STT)
- **静的配信対応** — GitHub Pages でそのままホスティング可能

### ファイル構成

```
├── index.html          # メインページ（SPA）
├── style.css           # アプリスタイル
├── app.js              # アプリケーションロジック
├── lib/                # 汎用ライブラリ（他プロジェクトでも再利用可）
│   ├── study-db.js         # IndexedDB 教材・進捗管理
│   ├── study-progress-ui.js    # 進捗可視化コンポーネント
│   └── study-progress-ui.css   # 進捗UI スタイル
├── samples/            # サンプル教材
│   └── greetings_10.tsv
└── guide/              # ユーザーガイド
    ├── user-guide.md
    └── tsv-format.md
```
